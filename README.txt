Como usar o site
================

Arquivos principais:
- index.html
- styles.css
- script.js
- assets/

Para publicar:
1. Vercel ou Netlify
   - Crie uma conta
   - Envie a pasta inteira site-felipe
   - O site entra no ar em poucos minutos

2. Hospedagem comum / cPanel
   - Envie todos os arquivos da pasta para a pasta public_html

3. Domínio recomendado
   - felipealvespatrimonio.com.br
   - felipeconsorciobrasil.com.br
   - felipealvesconsultoria.com.br

Melhor uso:
- Coloque o link do site na bio do Instagram
- Use o botão do WhatsApp como principal CTA
- Depois adicione depoimentos autorizados de clientes e prints de resultados
